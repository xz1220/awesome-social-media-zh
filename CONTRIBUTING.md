# 贡献指南

感谢补充中文社交媒体与自媒体运营资源。

## 基本原则

- 中文说明优先。
- 不收录刷量、违规采集、侵犯隐私或规避平台规则的资源。
- 不编造工具能力、价格、指标、平台规则或案例表现。
- 不提交未经授权的私密截图、聊天记录、后台数据或用户信息。
- 推荐资源时说明适用场景和限制，不只贴链接。

## 推荐格式

工具类资源：

```markdown
### 工具名

- 链接：
- 类型：
- 适用场景：
- 中文适配：
- 免费/付费：
- 优点：
- 限制：
- 风险：
- 观察日期：
```

案例类资源：

```markdown
### 账号或内容名称

- 平台：
- 链接：
- 观察日期：
- 值得学习：
- 公开可见指标：
- 不适合照搬的点：
```

## 提交流程

1. 找到最匹配的目录。
2. 按模板补充条目。
3. 确认链接可访问。
4. 确认没有包含私密信息。
5. 提交 Pull Request。
