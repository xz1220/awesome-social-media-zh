# awesome-social-media-zh

中文社交媒体与自媒体运营资源清单。关注工具、平台机制、内容生产、发布分发、数据复盘和可学习案例。

这个项目先服务一个真实需求：系统整理中文创作者、自媒体新手、独立开发者和小团队会实际用到的运营资源。所有条目都应该尽量说明适用场景，而不是只堆链接。

## 项目定位

- 中文优先：面向中文内容生态和中文使用者。
- 实用优先：优先收录能用于真实选题、生产、发布、复盘的资源。
- 场景优先：按任务和平台组织，而不是按工具名堆砌。
- 可验证优先：工具能力、平台数据和案例观察尽量标注来源与观察日期。

## 收录标准

一个资源更适合被收录，如果它满足以下条件之一：

- 能解决自媒体运营中的明确问题，例如选题、脚本、封面、剪辑、发布、复盘。
- 对中文平台或中文创作者有明显适配价值。
- 有清晰文档、公开主页、仓库、价格页或可验证来源。
- 已被真实使用过，并有明确的优点、限制和风险说明。
- 是值得学习的平台机制、账号案例、内容结构或运营方法。

暂不优先收录：

- 只有营销话术、没有可验证功能说明的工具。
- 需要高风险账号操作、刷量、采集隐私数据或规避平台规则的资源。
- 没有明确用途的泛泛资料。
- 只适合单个私有项目、无法泛化给其他创作者的内容。
